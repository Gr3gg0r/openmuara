# OpenMuara Examples

This directory contains the official, maintained example for OpenMuara.

## Checkout store

| Example | What it demonstrates | Run |
|---|---|---|
| [Checkout store](checkout-store/) | Landing page + checkout SPA with Fawry & Stripe + Mailpit | See [`checkout-store/README.md`](checkout-store/) |

The checkout store is the only example kept in the repo; it is actively tested
and should always work against the current OpenMuara release.
