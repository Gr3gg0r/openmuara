# Glossary

| Term | Definition |
|------|------------|
| **ARIA** | Accessible Rich Internet Applications; attributes that supplement HTML semantics for assistive technologies. |
| **Focus trap** | Keeping keyboard focus inside a modal or dialog until it is closed. |
| **Live region** | An element with `aria-live` that screen readers announce when its content changes. |
| **Roving tab index** | A pattern where only one item in a group (e.g., tabs) is in the tab order at a time, and arrow keys move focus within the group. |
| **Semantic HTML** | Using the right HTML element for its meaning (e.g., `<button>` for actions, `<main>` for primary content). |
| **Vanilla CSS** | CSS without a framework or preprocessor. |
