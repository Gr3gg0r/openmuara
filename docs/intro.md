# OpenMuara Documentation

OpenMuara is a local-first payment virtualization layer. It emulates payment providers such as Stripe, Fawry, SenangPay, iPay88, Billplz, and ToyyibPay so you can build and test billing flows offline, fast, and headlessly.

## Getting started

- **[Quickstart](quickstart)** — install, configure, and run your first emulated charge.
- **[Architecture](architecture)** — understand how requests flow through OpenMuara.
- **[Providers](providers)** — pick the provider you want to emulate.
